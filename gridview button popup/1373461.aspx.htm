     I think you can put some javascript such as ModalPopupExtender1.Show() in the OnClientClick attribute.

    <asp:ButtonField ButtonType="Image" ImageUrl="~/_Theme/images/redx.gif" CommandName="DeleteEntry" Text="Delete" OnClientClick="ModalPopupExtender1.Show()" ItemStyle-Width="40"/>

    I don't know why you want to use a popup for this and maybe you can tell me but I would rather use a message box for this purpose. Here is some code to use C# and VB. 

    C#

    public void Tester() 
{ 
    string he_say = null; 
    MsgBoxResult resulter = Interaction.MsgBox("Are you sure?", MsgBoxStyle.YesNoCancel); 
    if (resulter == MsgBoxResult.No) { 
        he_say = "No"; 
    } 
    else if (resulter == MsgBoxResult.Cancel) { 
        he_say = "Cancel"; 
    } 
    else { 
        he_say = "OK"; 
    } 
    Interaction.MsgBox(he_say); 
} 
    protected void ImageButton1_Click(object sender, System.Web.UI.ImageClickEventArgs e) 
    { 
        Tester(); 
    } 
    VB
        Public Sub Tester()
        Dim he_say As String
        Dim resulter As MsgBoxResult = MsgBox("Are you sure?", MsgBoxStyle.YesNoCancel)
        If resulter = MsgBoxResult.No Then
            he_say = "No"
        ElseIf resulter = MsgBoxResult.Cancel Then
            he_say = "Cancel"
        Else
            he_say = "OK"
        End If
        MsgBox(he_say)
    End Sub

     
    Protected Sub ImageButton1_Click(ByVal sender As Object, ByVal e As System.Web.UI.ImageClickEventArgs) Handles ImageButton1.Click
        Tester()
    End Sub
    I must just add that the code was originally written in VB.Net and used a converter to convert to C#.
    http://www.developerfusion.com/tools/convert/vb-to-csharp/

    We just use this in our web applications

    1    <asp:ImageButton ID="lnkDelete" runat="server" ImageUrl="~/img/i.p.delete.gif" CommandArgument='<%# Eval("Id") %>'
2    ToolTip="Delete Document" Style="cursor: pointer;" CommandName="DeleteDocument"
3    OnCommand="lnkDeleteDocument_Command" OnClientClick="return confirm('Are you certain you want to delete this document?');"
4    Visible='True'>

     

    We have to handle our deletes in code behind due to our using NHibernate as a datasource.  But this serves us with a confirm box for deletes.  if Ok isn't pressed, the code behind delete method doesn't get called.

    You don't need to show a modal popup when a simple javascript confirm box will work.  You are more than welcome to use a ModalPopup for this but it is overkill.

    To have a button that deletes and pops up a box the user has to click "OK" on before the delete takes place, all you have to do is use something like this for your delete button.

    <asp:TemplateField ShowHeader="False">

    <ItemTemplate>

    <asp:LinkButton ID="LinkButton1" runat="server" CausesValidation="False"

    CommandName="Delete" Text="Delete" OnClientClick="return confirm('Are you certain you want to delete this record?');"></asp:LinkButton>

    </ItemTemplate>

    </asp:TemplateField>

    Ted
