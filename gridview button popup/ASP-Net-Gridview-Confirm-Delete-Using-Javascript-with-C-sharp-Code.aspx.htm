In ASP.Net GridView control you can use Javascript code to validate the data or display the delete confirm and save changes confirmation messages for GridView fields and controls, to the user for confirming the action taken by the user. These JavaScript confirm messages return true if you click on OK button and return false when you click on CANCEL button.

Result of True or False return of JavaScript Confirm:

When user clicks on CANCEL button of javascript confirm box then it return false that stops the form from submitting the data.
Whereas when user clicks on the OK button of Javascript confirm box then it returns true that allows to post and submit the form data.

How to use JavaScript confirm with ASP.Net GridView Control?

ASP.Net GridView control has some template fields to that can add the delete command to the control e.g.: Command Field, Hyperlink Field, Button Field and Template Field. Among these fields you can use the Template field for easy implementation of JavaScript code with asp button control. Now you will think that why we are using Template field instead of other fields available to bind the delete command. Your answer is in below HTML source code of Gridview control.

HTML source code for GridView Control


<asp:GridView
    ID="GridView1"
    runat="server"
    AutoGenerateColumns="False"
    OnRowDeleting="GridView1_RowDeleting">
<Columns>

<asp:BoundField DataField="productid" SortExpression="productid" HeaderText="Product ID" />

<asp:BoundField DataField="productname" SortExpression="productname" HeaderText="Product" />

<asp:BoundField DataField="unitprice" SortExpression="unitprice" HeaderText="Price" />

<asp:TemplateField HeaderText="Delete">
<ItemTemplate>

<asp:LinkButton
    ID="lnkDelete"
    runat="server"
    CommandName="Delete"
    Text="Delete"
    OnClientClick="javascript : return confirm('Do you really want to \ndelete the item?');">
</asp:LinkButton>

</ItemTemplate>
</asp:TemplateField>

</Columns>

</asp:GridView>

Notice the code for link button control in itemTemplate of TemplateField. CommandName="Delete" will work for RowDeleting event of ASP.Net GridView control. You can write your C# code in RowDeleting event function of Gridview control 

Now take a look on other attribute of link button i.e. OnClientClick.
This event will handle the client end script i.e. JavaScript to confirm the user action. Use the following code for javascript confirm:

javascript:return confirm('Do you really want to \ndelete the item?');

// SQL Delete Command
